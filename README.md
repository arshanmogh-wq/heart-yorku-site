# HEART York University Site

Marketing site for HEART (Health Equity Advocacy Response Team), a York University student-led organization focused on health equity through advocacy, education, outreach, and community partnerships.

## Stack

- Astro 5
- Static output
- `@astrojs/sitemap`

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Project Structure

```text
/
├── public/        # Static assets, favicons, verification files
├── src/
│   ├── components/
│   ├── layouts/
│   ├── pages/
│   └── styles/
├── astro.config.mjs
└── package.json
```

## Deployment

The site is configured for static output and includes sitemap generation via Astro.
